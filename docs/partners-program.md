## Sponsorship Tiers

| Tier | Monthly | Benefits |
|------|---------|----------|
| 🥉 Bronze | $50 | Logo on README, monthly thank-you in release notes |

## Partner Types

| Type | Description | Example |

| **Infrastructure Partner** | Provides hosting/CI/CD resources | Cloud providers, CI platform vendors |
